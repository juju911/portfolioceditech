LAB 8-9 — Créer et Déployer un Site Vitrine (HTML & CSS) avec AWS Amplify
Objectif :
Créer un site vitrine pour présenter votre parcours ou piste de spécialisation (ex. :
Développement Web, Cloud, IA, etc.), en utilisant uniquement HTML et CSS.
Le site doit inclure :
• Un logo
• Un menu de navigation (Accueil, Contacts)
• Des images illustratives
• Du texte explicatif
• Un design simple et harmonieux (soyez créatif)
1. Créer votre site
Il doit contenir :
o index.html
o style.css
o Un dossier images pour votre logo et vos visuels.
2. Déploiement : AWS Amplify
o Connectez votre dépôt GitHub.
o Déployez et récupérez l’URL.
